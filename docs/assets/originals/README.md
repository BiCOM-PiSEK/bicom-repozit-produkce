# docs/assets/originals/

Tato složka obsahuje **zdrojové vizuály v plném rozlišení**.

## Pravidla
- Pojmenování: `kebab-case`, bez diakritiky
- Max velikost jednoho souboru: 5 MB
- Formáty: SVG (vektory), PNG (rastrové podklady), JPEG (fotky)
- Optimalizované verze pro web se exportují do `public/assets/img/`
- Viz `docs/ASSET_STRATEGY.md` pro kompletní workflow

## Struktura
```
originals/
├── logo/         — SVG logo zdroje, wordmark, icon
├── hero/         — hero vizuály v plné kvalitě
├── og/           — zdrojový OG obrázek (1200×630+)
├── icons/        — zdrojový SVG pro favicon sadu
├── gallery/      — fotky ordinace, Lenka portrét
└── certificates/ — skeny/fotky certifikátů
```
